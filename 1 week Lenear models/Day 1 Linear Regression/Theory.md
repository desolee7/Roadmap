# День 1: Линейная регрессия

## 1. Что такое линейная регрессия

Линейная регрессия — это модель, которая описывает зависимость целевой переменной от признаков как линейную комбинацию.

Идея:
y ≈ ŷ = w₀ + w₁x₁ + w₂x₂ + ... + wₖxₖ

В векторной форме:
ŷ = Xw

где:
- X — матрица признаков (n × k)
- w — вектор коэффициентов (k × 1)

---

## 2. Как считается (обучение модели)

Цель — найти такие коэффициенты w, которые минимизируют ошибку между предсказаниями и реальными значениями.

### Нормальное уравнение (аналитическое решение)

w = (XᵀX)^(-1) Xᵀy

Условия:
- XᵀX должна быть обратимой
- работает хорошо на небольших данных

**Примеры в Python:**

```python
# sklearn
from sklearn.linear_model import LinearRegression

model = LinearRegression()
model.fit(X_train, y_train)
y_pred = model.predict(X_test)


---

### Градиентный спуск (итеративный метод)

Обновление весов:

w := w - η ∇L(w)

где:
- η — learning rate
- L(w) — функция потерь

Используется при больших данных
# sklearn - стохастический градиентный спуск
from sklearn.linear_model import SGDRegressor

## 3. Функция ошибки (loss function)

Чаще всего используется MSE:

MSE = (1/n) * Σ (y_i - ŷ_i)^2

Почему:
- дифференцируема
- выпуклая
- даёт аналитическое решение
- сильно штрафует большие ошибки

Альтернативы:
- MAE — менее чувствительна к выбросам
- Huber loss — компромисс

---

## 4. Предположения линейной регрессии

1. Линейность зависимости  
2. Независимость ошибок  
3. Гомоскедастичность (одинаковая дисперсия ошибок)  
4. Нормальность ошибок (важно для статистики, не для предсказаний)  
5. Отсутствие мультиколлинеарности  

---

## 5. Мультиколлинеарность

Сильная корреляция между признаками.

Проблемы:
- нестабильные коэффициенты  
- невозможность обратить XᵀX  
- переобучение  

Решения:
- удалить признаки  
- регуляризация (Ridge, Lasso)  
- PCA  

---

## 6. Метрики качества

### R²

R² = 1 - (SS_res / SS_tot)

Показывает долю объяснённой дисперсии

---

### Adjusted R²

R²_adj = 1 - (1 - R²) * (n - 1) / (n - k - 1)

Штрафует за лишние признаки
